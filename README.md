## use

#### 安装tingle-style子模块

添加`tingle`的基础样式模块，在根目录下，执行下面的命令，通过git的子模块方式，把依赖加载进来。

```shell
git submodule add git@gitlab.alibaba-inc.com:alinwmobile/tingle-style.git tingle/tingle-style
```

#### 启动

```
npm start
```
